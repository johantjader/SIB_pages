# SIB Normativa Artefakter

## What This Is

En versionshanterad och högtillgänglig webbplats byggd med Docusaurus för att publicera normativa artefakter för en behörighets- och identitetsfederation. Projektet syftar till att flytta existerande material från en sandbox-miljö till en produktionsklar lösning som hostas på GitHub Pages.

## Core Value

En tillförlitlig, versionshanterad och tillgänglig källa för normativa artefakter för alla medlemmar i federationen.
