# Utgått: Denna artefakt har ersatts

Detta dokument har ersatts av **[Deklaration om efterlevnad för Federationsmedlem](deklarationsmall_for_federationsmedlem.md)**.

Den nya mallen är en konsoliderad och enhetlig version som gäller för alla organisationer som ansluter sig till federationen i rollen som Federationsmedlem, oavsett om de agerar som förlitande part, utfärdare av behörighetshandling, eller bådadera.

**Vänligen referera till och använd den nya artefakten istället.**
